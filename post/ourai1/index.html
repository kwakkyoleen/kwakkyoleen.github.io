모두를 위한 딥러닝-딥러닝의 기본 1. 머신러닝의 개념과 용어

January 18, 2020

1. 기본적인 머신러닝 용어와 개념 설명

머신러닝

  • explict program 의 제한을 벗어난 프로그램. 개발자가 정해준 일정한 로직을 따르지 않고 학습한 내용을 토대로 움직이는 프로그램.

머신러닝의 종류

  • supervised learning : training set(레이블들이 정해져 있는 데이터)을 가지고 학습을 하는 방법 - 예시) 개와 고양이를 구분하는 프로그램, 개와 고양이로 레이블이 구분된 자료로 학습
  • unsupervied learning : un-labeled data(데이터가 레이블이 주어지지 않은 경우)를 보고 유사한것들을 구분하여 학습하는 방법 - 예시) 구글 뉴스의 그룹화
  • 이번 강의에서는 supervised learning을 중점으로 볼것임

ML에서의 일반적이 문제 유형

  • 레이블링된 이미지를 보고 학습
  • 스펨메일로 구분된, 또는 구분되지 않은 메일을 보고 학습
  • 시험 준비 시간과 시험 점수가 주어진 데이터를 보고 학습

training data set

  • 답이라고 할 수 있는 Y값과 특징이라고 할 수 있는 X를 가지고 있는 레이블화된 데이터.
  • 이 데이터를 통해 학습한 모델이 생성
  • 모델에 임의의 textX를 넣어주었을 때 예측된 값 Y를 출력
  • 따라서 trainging data set은 반드시 필요하다

supervised learning의 타입

  • regression : 특정한 법위에서 값을 예측 - 예시) 공부 시간에 따른 점수 예측 (0 ~ 100 사이의 값 예측)

data set 예시

X(시간) Y(점수)
10 90
9 80
3 50
2 30

위 데이터 셋을 기반으로 학습하여 학습 시간(X)을 넣으면 점수를 예측

  • binary classfication(이진 분류) : True or False 두가지로 구분 - 예시) 공부 시간에 따른 pass or non-pass예측

data set 예시

X(시간) Y(P/F)
10 P
9 P
3 F
2 F

위 데이터 셋을 기반으로 학습하여 학습 시간(X)을 넣으면 P/F 예측

  • multi-label classification : 여러가지 경우로 구분 - 예시) 공부 시간에 따른 등급예측

data set 예시

X(시간) Y(등급)
10 A
9 B
3 D
2 F

위 데이터 셋을 기반으로 학습하여 학습 시간(X)을 넣으면 등급 예측

1-L. TenserFlow의 기본

TenserFlow에대한 간략한 설명

  • 머신러닝을 위해 구글에서 만든 오픈소스 라이브러리
  • 머신러닝을 위한 라이브러리로 텐서플로우를 가장 많이 씀 -> 공부하기 쉽고 미리 만들어진 소스코드가 풍부함
  • data flow graphs를 이용해서 numerical한 계산을 할 수 있음

Data flow graph가 무엇인가?

  • 그래프 : 노드와 노드간을 연결하는 엣지
  • 노드(operation)들이 엣지(데이터)의 배열(temsors)을 받아 계산을 하는 흐름에서 어떤 연산이 일어나 원하는 결과물을 얻거나 어떤 작업을 하는 것.

Tenser flow 설치하기

  • pip instal –upgrade tensorflow 로 설치 가능
  • gpu가 있다면 pip install –upgrade tensorflow-gpu
  • 설치 후 import tensorflow as tf 로 tensorflow import
  • tf.version 실행. 설치 되었는지 확인

TensorFlow Hello World!

다음을 실행

import tensorflow as tf

hello = tf.constant("hello, tensorflow!")

sess = tf.Session()

print(sess.run(hello))

기존 tensorflow에서는 constant라는 노드를 만들고 Session을 생성시킨 다음 session에 노드를 집어넣어 run 시킴으로써 계산을 진행하였다.

하지만 tensorflow 2.0 버전에선 Session이 사라지고 즉시 실행으로 바뀌었기 때문에 이 코드로는 출력이 불가능 하다.

import tensorflow as tf

hello = tf.constant("hello, tensorflow!")

print(hello.numpy())

대신 hello 라는 오퍼레이터를 numpy 속성으로 변환시켜 출력하였다.

b'hello, tensorflow!'

라는 출력이 나왔다. 여기에서 b는 Byte literals를 뜻한다.

노드를 추가한 조금 더 복잡한 코드를 실행시켜보았다.

import tensorflow as tf

node1 = tf.constant(3.0,tf.float32)
node2 = tf.constant(4.0)
node3 = tf.add(node1, node2)

print("node1:",node1,"node2",node2)
print("node3:",node3)

sess = tf.Session()
print("sess.run(node1,node2): ",sess.run([node1, node2]))
print("sess.run(node3): ", sess.run(node3))

하지만 마찬가지의 이유로 실행이 되지 않기에 위와같은 방법으로 코드를 수정해 실행하였다.

import tensorflow as tf

node1 = tf.constant(3.0,tf.float32) #node1을 만들고
node2 = tf.constant(4.0) #node2 도 만든다
node3 = tf.add(node1, node2) #더하기 노드를 만들고 node1과 node2를 연결한다

print("node1:",node1,"node2",node2)
print("node3:",node3)

print("[node1.numpy(), node2.numpy()]: ",[node1.numpy(),node2.numpy()])
print("node3.numpy(): ",node3.numpy())

기존의 tensorflow와 달리 Session을 거치지 않고 바로 실행되는 모습을 볼 수 있다.

placehold

a = tf.placeholder(tf.float32)
b = tf.placeholder(tf.float32)
adder_node = a + b

print(sess.run(adder_node, feed_dict={a: 3, b: 4.5}))
print(sess.run(adder_node, feed_dict={a: [1,3], b: [2,4]}))

결과

7.5
[3.  7.]

placeholder를 사용하면 sess로 계산을 진행하는 단계에서 feed_dict로 값을 지정해줄 수 있다. 다만 tensorflow 2.0에서는 Session이 사라졌기 때문에 위와 같은 코드는 불가능 하다.

@tf.function
def addfun(a, b):
    return a+b

adder_node = addfun(3, 4.5)

print(adder_node)

대신 funtion데코레이션이 추가되어 비슷하게 사용할 수 있다.

텐서플로우의 동작 원리

  1. 텐서플로우 operation을 사용하는 그래프를 만든다.
  2. 데이터를 넣고 그래프를 실행시킨다.
  3. 그래프 안의 변수들을 업데이트 시킨다.(그리고 값을 반환한다.)
